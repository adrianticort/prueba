## ✨ Características Principales

Este proyecto está construido como una *Single Page Application (SPA)*, lo que significa que es rápido y fluido, sin necesidad de recargar la página.

*   **📅 Calendario Interactivo:** El centro de mando de la aplicación. Permite seleccionar un día para ver las opciones disponibles.
*   **🏋️‍♂️ Gestión de Rutinas:** Visualización detallada de los ejercicios diarios, incluyendo series, repeticiones, consejos e imágenes de ejemplo.
*   **🍏 Plan Nutricional:** Recetas de desayunos saludables para cada día de la semana.
*   **📊 Seguimiento de Progreso:**
    *   **Persistencia de datos:** Tu progreso se guarda en el navegador (`localStorage`), así que no se pierde al cerrar la pestaña.
    *   **Estadísticas visuales:** Una gráfica de barras muestra tu consistencia semanal.
    *   **Métricas clave:** Tarjetas que muestran tu racha actual de entrenamiento, el total de sesiones completadas y tu mejor semana.
*   **🎨 Diseño Moderno y Responsivo:** La interfaz está diseñada para ser atractiva y funcional tanto en escritorio como en dispositivos móviles.

## 🛠️ Tecnologías Utilizadas

Este proyecto está construido enteramente con tecnologías web estándar, sin necesidad de frameworks complejos, para maximizar la velocidad y la simplicidad.

*   **HTML5:** Para la estructura semántica del contenido.
*   **CSS3:** Para todo el diseño visual, incluyendo:
    *   **Variables CSS:** Para un tema de colores fácil de mantener.
    *   **Flexbox y Grid:** Para la maquetación moderna y responsiva.
    *   **Animaciones y Transiciones:** Para una experiencia de usuario fluida.
*   **JavaScript (ES6+):** Para toda la lógica de la aplicación:
    *   Manipulación del DOM.
    *   Gestión de eventos.
    *   Persistencia de datos con `localStorage`.
